# Mr. Pasta

Authentic pasta delivered to your doorstep.

## Project Structure

- `src/assets/`: Logos, fonts, and images.
- `src/components/`: Reusable React components (Buttons, Card, etc.).
- `src/pages/`: Page-level components (Home, Products, etc.).
- `src/utils/`: Helper functions (WhatsApp integration, cart logic).

## Setup Instructions

1. **Clone the repository.**
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```

## Firebase Setup

Update `src/firebase/firebaseConfig.js` with your Firebase project credentials.
